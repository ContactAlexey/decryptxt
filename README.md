<h1 style="color: #4B0082;">üîì Decryptxt</h1>

<p>
    <span style="background-color:#3572A5; color:white; padding:3px 8px; border-radius:12px;">Python 3.x</span>
    <span style="background-color:#28a745; color:white; padding:3px 8px; border-radius:12px;">MIT License</span>
    <span style="background-color:#d73a49; color:white; padding:3px 8px; border-radius:12px;">Kali Linux</span>
</p>

<p>
    <strong>Decryptxt</strong> is a decryption tool for Kali Linux designed to
    <strong>restore original text files encrypted with <a href="https://github.com/ContactAlexey/decryptxt/blob/main/LICENSE](https://github.com/ContactAlexey/cryptxt" target="_blank">Cryptxt</a>.
    It reverses the encryption process by decoding the binary values and applying
    the stored key used during encryption.
</p>

<h2 style="color:#4B0082;">‚ú® Features</h2>
<ul>
    <li>Decrypt files generated by <strong>Cryptxt</strong>.</li>
    <li>Automatically extracts the encryption number from the file.</li>
    <li>Converts multiplied binary values back to ASCII characters.</li>
    <li>Restores the original readable text.</li>
    <li>Simple command-line usage in Kali Linux.</li>
</ul>

<h2 style="color:#4B0082;">üöÄ Installation</h2>
<p>Clone the repository and install Decryptxt:</p>

<pre style="background-color:#f0f0f0; padding:10px; border-radius:5px;">
<code>git clone https://github.com/ContactAlexey/decryptxt.git
cd decryptxt
chmod +x install.sh
./install.sh</code>
</pre>

<p>This will install <strong>decryptxt</strong> and allow it to be executed from anywhere in the terminal.</p>

<h2 style="color:#4B0082;">üñ•Ô∏è Usage</h2>

<p>1. Navigate to the directory containing the encrypted file.</p>
<p>2. Run:</p>

<pre style="background-color:#f0f0f0; padding:10px; border-radius:5px;">
<code>decryptxt &lt;encrypted_file.txt&gt;</code>
</pre>

<p>3. The decrypted content will be written to a new readable text file.</p>

<h2 style="color:#4B0082;">‚öôÔ∏è How it works</h2>
<ol>
    <li>The encrypted file is read.</li>
    <li>The last number (encryption key) is extracted.</li>
    <li>Each numeric value is divided by the key.</li>
    <li>The result is converted from binary to ASCII.</li>
    <li>The original text is reconstructed and saved.</li>
</ol>

<p>The tool assumes the file structure generated by Cryptxt:</p>

<pre style="background-color:#f0f0f0; padding:10px; border-radius:5px;">
<code>number number number ... &lt;encryption_key&gt;</code>
</pre>

<h2 style="color:#4B0082;">üìÇ Example</h2>
<details>
  <summary>Show example</summary>

  <p>Encrypted file <code>hidden_message.txt</code>:</p>
  <pre style="background-color:#f9f9f9; padding:10px; border-radius:5px;">
<code>696 1056 1856 2112 528 1584 2112 2112 1056 1104 2112 42</code>
  </pre>

  <p>Command:</p>
  <pre style="background-color:#f9f9f9; padding:10px; border-radius:5px;">
<code>decryptxt hidden_message.txt</code>
  </pre>

  <p>Output:</p>
  <pre style="background-color:#f9f9f9; padding:10px; border-radius:5px;">
<code>Hello World</code>
  </pre>

  <p><em>Note:</em> The value <code>42</code> is used internally to reverse the encryption.</p>
</details>

<h2 style="color:#4B0082;">‚ö†Ô∏è Important</h2>
<p>
    Decryptxt only works with files generated by Cryptxt.
    It is not intended to break or bypass encryption,
    but to correctly reverse a custom encoding system.
</p>
